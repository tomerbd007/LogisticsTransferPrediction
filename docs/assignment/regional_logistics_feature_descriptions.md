# Regional Logistics Transfers Dataset - Feature Descriptions

This document describes the features included in the Regional Logistics Transfers Dataset.

| Feature Name | Description |
|---|---|
| transfer_id | Unique transfer identifier. Intended for record identification and prediction submission purposes, and not necessarily recommended for direct use as a predictive feature. |
| operation_date | Date on which the transfer operation occurred. |
| day_of_week | Day of the week on which the transfer was performed. |
| scheduled_dispatch_time | Scheduled dispatch time of the transfer in HHMM format. |
| expected_dispatch_delay_minutes | Estimated dispatch delay generated from operational planning systems and historical information. |
| scheduled_arrival_time | Scheduled arrival time at the destination facility in HHMM format. |
| source_facility | Anonymous identifier of the source facility. |
| source_region | Anonymous identifier of the source region. |
| destination_facility | Anonymous identifier of the destination facility. |
| destination_region | Anonymous identifier of the destination region. |
| logistics_provider | Anonymous or encoded identifier of the logistics provider responsible for the transfer. |
| scheduled_travel_time_minutes | Scheduled transfer duration in minutes. |
| route_distance_miles | Estimated transfer route distance in miles. |
| expected_provider_delay_minutes | Estimated delay associated with provider-related operational factors. |
| weather_disruption_minutes | Delay in minutes attributed to weather-related disruptions. |
| network_congestion_delay_minutes | Delay in minutes attributed to congestion in the logistics network. |
| expected_inspection_delay_minutes | Estimated delay associated with inspections, controls, or approval procedures. |
| expected_weather_disruption_minutes | Estimated delay associated with weather-related disruptions. |
| expected_network_congestion_delay_minutes | Estimated delay associated with congestion within the logistics network. |
| weather_congestion_index | Combined index representing the interaction between weather disruptions and network congestion. |
| external_system_score | Score generated by an external operational system. A large portion of the values are intentionally missing. |
| internal_metric_a | Anonymous internal operational metric. |
| internal_metric_b | Anonymous internal operational metric. |
| internal_timestamp_a | Anonymous internal operational timestamp. |
| internal_timestamp_b | Anonymous internal operational timestamp. |
| internal_operational_score | Internal operational score generated for analytical purposes. |
| is_delayed | Target variable. A value of 1 indicates that the transfer was delayed by more than 30 minutes, while 0 indicates that the transfer was not delayed by more than 30 minutes. |
