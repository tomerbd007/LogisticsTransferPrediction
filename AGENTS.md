# Agent Guide

Before editing, read `INSTRUCTIONS.md`. It is the canonical guidance for this project.

Key rules:

- Keep the solution appropriate for a BSc Introduction to Machine Learning project.
- Keep required runnable implementation code inside `main.ipynb`.
- Preserve course-provided data file names.
- Use `data/` for train/test CSV files.
- Use `docs/assignment/` for assignment files and `docs/course_materials/` for class material.
- Avoid unrelated rewrites, broad refactors, and formatting churn.
- Do not commit generated caches, notebook checkpoints, virtual environments, or intermediate outputs.
- Coordinate carefully around notebook edits because JSON notebooks are difficult to merge.
